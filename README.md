# Sequencer Freelancer Hackathon

## Prerequisites

- [MongoDB](https://www.mongodb.org/downloads)
- [Node.js 6.0+](http://nodejs.org)

## Getting Started

```bash
# Install NPM dependencies
npm install

# Then simply start your app
node app.js
```
